How do I backorder a domain?

Start by searching for a domain in the search box above. You will get results that show whether the domain is available for purchase, or if it's owned by someone. If it is owned, you can create a backorder request by clicking 'Reserve this Domain".

Next, you will be prompted to enter your card info to save the reservation. Don't worry, we won't chart your card unless we actually are able to get the domain for you.


How will I know if I got the domain?

We'll let you know if we were able to get the domain for you, and automatically charge your card when that happens.


What if someone else backorders the same domain that I want?

For the time being, reservations are first come, first serve.
In the future, we plan to add the ability to bid on domains that are requested by multiple people.
